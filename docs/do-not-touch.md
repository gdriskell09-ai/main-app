# Do Not Touch Without Approval

## Repo Boundaries

- Do not touch `scrub-club-app` when working in `main-site`.
- Do not copy schema, routes, or app patterns between repos without explicit approval.

## Forbidden For Now

- payments
- final pricing
- Stripe
- subscriptions
- real legal advice
- binding contracts
- e-signatures
- QuickBooks
- accounting/tax integrations
- marketplace checkout
- OpenRouter/provider routing
- BYO API key storage
- automatic AI API calls
- AI vision/photo quoting
- map measuring
- website importing
- dropshipping/storefront engine
- full auth rewrite
- database schema rewrite
- full app rewrite
- full visual builder replacement

## Sensitive Areas

Ask before editing:

- auth/session flow
- Supabase schema files
- package dependencies
- lockfiles
- `proxy.ts`
- API routes
- invoice/payment-related code
- map/geospatial code
- generated website renderer architecture

## Product Guardrails

- No hard-coded final pricing.
- No automatic AI calls.
- No client-side secrets.
- No legal/financial advice claims.
- No publishing generated content without owner review.
